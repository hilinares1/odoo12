How to use this module

Step 1 : Dashboard

  • Go to menu Leaves / Dashboard

    • The information contains Leave Type, Year, Type, Parent, Maximum Allowed, Leaves Already Taken, and Remaining Leaves

    Step 2 : Create Leave Types

  • Go to menu Leaves / Configuration / Leave Type
  • To create the leaves, fill the information :
    • Leave Type : name of the leaves
    Apply Double Validation : Check if True

    Allow Override Limit : Allow to take leaves more than allocated (check if True)

    Year : Leaves Year
    Period In : When you create leaves
    Period All : When Leaves type is valid
    Step 3 : Allocate Leaves

  • Go to menu Leaves / Leaves to Approve / Leaves Allocation
    • Step 4 : Create Leaves

  • Go to menu Leaves / My Leaves / Leaves Request
  • Click on Create button
  • Fill Description
  • Select Leave Type
  • Select Duration

    • Notes :

    (depends to your working schedule)
    Morning : 08:00 - 12:00
    Afternoon : 03:00 - 17:00
  • click Submit button
