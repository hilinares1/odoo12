Tax Comment

Add Tax Comment on printed invoice

Falinwa Limited

Story behind the features

Odoo behavior : Odoo does not propose to add comment based on taxes.

Falinwa behavior : if you have due to local regulation, mandatory sentences to print on your invoices according to the tax you use, you can configure this comment on the tax and it will automatically print out on the invoice.

Module + : this module ensures to provide legal invoices including mandatory references which are often forgotten if it's not automatized.

Falinwa recommendation :

  • Installation : required in Europe, at least for intracom transaction, if you issue invoices.

How To Do It

Step 1
  • Go to Accounting, Configuration > Accounting > Tax Comment
  • Create Tax Comment
    • Step 2
  • Go to Accounting, Configuration > Accounting > Taxes
  • Choose 1 tax (ie : Tax 15%)
  • on tab Advance Option, select the comment on field Comment to print on customer invoice
  • Save
    • Step 3
  • Create an Invoice with lines, put a tax Tax 15%
  • Save the Invoice
  • Print the Invoice
    • Step 4
  • Tax comment will show on the invoice
