Story behind the features

Odoo behavior : when you create an user, Odoo will automatically create a related partner but which is an individual not related to any company

Falinwa behavior : at creation, the user is directly related to its current company.

Module + : as the user is properly created from the beginning, configuration of user, employee, work partner and private partner is much more intuitive and avoid recurrent configuration mistakes.
