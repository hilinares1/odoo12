Odoo document system is clear and comfortable to use. However, it is not designed to work with files as specific cloud storage solutions like OneDrive, GoogleDrive, OwnCloud, and DropBox do. To process attachments users need to download a file, to change it, and to upload back. Documents are not synced locally and Odoo doesn't have powerful previewers and editors. Specific cloud storage clients don't have such disadvantages. This tool is a technical core to provide logic of automatic and bilateral files synchronisation between Odoo and your cloud storage solution.

The tool is compatible with both Odoo Enterprise and Odoo Community.
