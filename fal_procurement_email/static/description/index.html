Procurement Request by Email

Request procurement by Email

Falinwa Limited

The Features

  • Configure email alias for procurement
  • Send email containing below format
    [fieldname:value]

    available fieldname:

    • partner_id => to specify vendor
      alternative key:
      • partner_id
      • partner
      • contact
      • vendor
      • supplier
    • product_id => to specify product
      alternative key:
      • product
      • product_id
      • fal_req_product_id
    • description => to set a description
      alternative key:
      • description
      • desc
      • fal_req_product_description
    • comment => to have a comment
      alternative key:
      • comment
      • comments
      • fal_warehouse_manager_comment
    • quantity => to set the quantity
      alternative key:
      • quantity
      • qty
      • fal_req_product_qty

    Example 

                
    
                [partner_id: Deco]
                [product_id: DEsk0004]
                [comment:For new employee]
                [description: the black one]
                [qty: 1]
