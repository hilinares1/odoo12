Three Levels Purchase Order Approval

This module is developed to set the three level approval of Purchase Order.

Company Configuration

If you want to use three steps approval, PO three steps approval is must be true.

Groups for the Approval


Go to purchase order menu, create RFQ / Purchase Order


If PO amount is greater than the company configured amount, It is go for Manager Approval. Set to Manager, Accountant Manager, PO/Director Manager.


PO Manager Approval Menu


When PO Manager is approve the order,PO go for the next approval state.


PO Accountant Approval Menu


When Accountant Manager is approve the order,PO go for the next approval state.


PO Director Approval Menu


Director Can Approve the PO or they can cancel the PO.
