How to use this module

Step 1 : Go to Accounting, Adviser > Project Budget and create a project budget
Step 2 : On Accounting, Configuration > Management > we can create Project Budget Tags
Step 3 : On Accounting, Reports > Management > Project Budgets line is created automatically from the project budget that we created before
Step 4 : Go to Project, create Project Budgets
Step 5 : Project Budget Line is created automatically based on Project Budget that we created before
Step 6 : If there is revision of project budget, you can create on Project Budget Revision form
