Story behind the features

Odoo behavior : Alias or Short Name is not available in Odoo. Odoo put only name in reference (code) as the only partner identity

Falinwa behavior : Many company famouse with its brand then its legal name. Short name on partner will help users to easily identify the partner.

Module + : strongly improve the management of your partner management.

Falinwa recommendation : we install this module on 100% of our database. Very usefull to recognize partners using short name and improve better tree view when the partner name is long.
