Story behind the features

Odoo behavior : in v12, Odoo proposes the batch payment but gives this option only from the payment list.

Falinwa behavior : you select all the invoices you want to pay, then make a batch payment. It will create both the payments and the batch linked to all the payments.

Module + : easier to create the batch from the invoices than from the payment.

Business case : every week, you want to issue a batch payment for your suppliers and export the related SEPA. Thanks to the batch number, once your invoices are paid, you can filter on the batch number and export the correct SEPA file. Without this batch number, you have no possibility to group the payment per batch.

Falinwa recommendation :

  • Installation : 100% of the database using SEPA payment or making the supplier payment per batch at specific dates.

How to use this module

Step 1 : Select the invoices
  • Select the Customer Invoices or
  • Select the Vendor Bills you want to pay
  • Go to Action and click on Register Multi Payment
    • Step 2 : Allocate the amount
  • On each line of invoice, change the payment amount if partial payment or
  • Click on Validate Button
    • Step 3 : Validate Payment
  • On the payment list, there are draft payment to validate
  • Validate the payment after you receive your money
