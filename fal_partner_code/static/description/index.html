Story behind the features

Odoo behavior : Odoo removed the notion of partner code, in order to focus on partner name.

Falinwa behavior : you have a field Internal Reference for your Partner Code and you can define an automatic sequence for your partner.

Module + : provide the 2 approaches - by name and by code. The code is the only way to ensure the uniqueness of the partner and so to control the quality of the database.

Falinwa recommendation :

  • Installation : 100% of the database.
  • Recommendation / Best practice : properly define your sequence when you use this module.

The Features

Features:

  1. Different Number Format for Supplier, Customer and Both
  2. Able to search partner using partner code
  3. Only Partner with type is a Company
