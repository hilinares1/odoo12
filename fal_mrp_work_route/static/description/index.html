How to use this module

Step 1 : Firstly, we must go to Manufacturing > Configuration > Routings & Planning, then choose Manage production by work orders, and Apply
Step 2 : Create a work center (Manufacturing > Master Data > Work Centers), if the work center is production place, choose type Routing Work Center
Step 3 : Create a work center, if the work center is production machine then choose type Workshop Machine
Step 4 : Create a production order
Step 5 : Confirm and Reserve Raw Material
Step 6 : Plan production order
Step 7 : Check on Menu Planning by Workshop Machine
Step 8 : Select Working Machine and Get Ready Work Orders
Step 9 : Group by Routing Work Center and Working Machine
Step 10 : Filter Routing Work Center by contains
Step 11 : Filter Working Machine by contains
Step 12 : Menu Planning by Routing Work Center and Planning by Workshop Machine
