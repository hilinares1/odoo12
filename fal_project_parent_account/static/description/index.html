Story behind the features

Odoo behavior : Odoo does not give the notion of parent on project and so the possibility to create a project structure.

Falinwa behavior : you can give a parent project and organize your whole project structure.

Module + : make a full control of your project structure organization and get the possibility to analyze your project from a macro point of view (ie the general one) or phase by phase (ie the subproject level).

Falinwa recommendation :

  • Installation : 100% of databases using project.
  • Recommendation / Best practice : a parent project should only be used to read data and not to record directly some moves on this project itself. Input are done on subprojects only. The project structure is the only way to keep a proper organization of all your projects.
