Odoo Product Pack Bundle

Product Pack/Bundle on Sales and Delivery

This module allow you to Create Pack / Bundle products and add it to Sales order and Delivery Orders.

Main Features:

  • Allow to create Pack / Bundle Products.
  • Allow set of products into the Pack / Bundle.
  • Allow you to quickly add Bundle into the Sale order.
  • Allow you to quickly add Bundle into the Delivery order.
  • For more details please see Video in Live Preview.
  • Old Video Link : https://youtu.be/gkU-pnHNUGc
  • New Video Link : https://www.youtube.com/watch?v=l62JLX9LAVw

Highlighted Steps:

Create Pack / Bundle Product:

Create Pack / Bundle Product->Tick "Is Bundled"->Add Bundle Products.

Create Sale order:

Create Sale order-> Open Wizard -> Select Bundle Product and add Quantity -> Add Pack / Bundle

Create Deliver order:

Create Delivery order-> Open Wizard -> Select Bundle Product and add Quantity -> Add Pack / Bundle

System Menus:

  • Sales
    • Sales
      • Bundle Products

Bundle Products Menu

Login with Sales User/Manager to Create Pack/Bundle Product.

Pack / Bundle Product

Tick "Is Bundled" product for declare it as Bundle Product.

Pack / Bundle Products tab on Bundle Products

Add Bundle Products as the products on Bundle product.

Create Sale Order

Create the Sale Order and / or Open Wizard from the "Add Pack / Bundle" Button

Select Bundle Product to add the bundle as the sellig products.
Bellow List of the products is the Products of the bundle.

Add The Pack / Bundle as the selling item and also add the Products of the Pack / Bundle.

Create Delivery Order

Create the Delivery Order and / or Open Wizard from the "Add Pack / Bundle" Button

Select Bundle Product to add the bundle as the picking products.
Bellow List of the products is the Products of the bundle.

Add The Pack / Bundle as the picking items and also add the Products of the Pack / Bundle.
