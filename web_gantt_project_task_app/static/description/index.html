Web Gantt View for Project Task in Odoo

This apps helps user to view web gantt view for project task in quarter day and half day.

Half Day View of Task.

User can see task by half day filter on gantt view.

Days Duration Between Assign Date to Deadline on Gantt view

User can see project task duration from assign date to deadline on gantt view.

Quarter Day View of Task.

User can see task by quarter day filter on gantt view.
