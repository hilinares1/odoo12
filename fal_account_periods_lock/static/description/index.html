Story behind the features

Odoo behavior : you have only 1 date to lock the accounting entries, which is quite restrictive, especially at the end of the fiscal year.

Falinwa behavior : we give the possibility to lock the period based on timeframe (like each month) and based on several level of access rights (accountant, adviser and with some extra option, employee for timesheet and expense).

Business case: if your fiscal year finihs on 31/12, you will work on it up to april in order to prepare the audit but you will already have to close the current year (january, february, march) each 10 of the next months to be sure daily records are properly done.

Falinwa recommendation :

  • Installation: install 100% of the database.
  • Recomendation: to be installed as soon as you have an accounting department with more than 1 user.

How to use this module

Create Period
  • On Period Lock menu, click create the period lock,
  • Select start date and end date
  • Click Create Monthly Periods,if period is monthly
  • or Create 3 Months Periods if the period is quarterly
  • after that the periods lock lines are created automatically
    • List of Periods
  • On Periods menu, the list of periods is created automatically
    • Check How it Works
  • To see how it works, on Customer Invoices, create a customer invoice with the date after adviser locking date then click validate, and we will see a warning "You cannot add/modify entries prior to and inclusive of the lock date" because we create the invoices after the locking date.
