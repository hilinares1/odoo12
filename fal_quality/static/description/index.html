Product Quality Control

Check quality product for each product movement

Why you need module ?

This module is to control quality of product you Receive, Deliver or Produce from manufacturing process
  • Quality Points : Parameter you want to check on product
  • Quality Check : is a form to fill when quality process is start
  • Quality Alerts : a warning when Quality Check founds fails

    • Definition

  • Quality Points : Parameter you want to check on product
  • Quality Check : is a form to fill when quality process is start
  • Quality Alerts : a warning when Quality Check founds fails

    • How to use this module :

    Step 1 : Create Quality Points Template
  • Go to menu Quality / Configuration / Quality Control Points Template
  • Select the product category
  • Select Operation
    Operation is process when we quality check needed
  • Select Operation

    • With Quality Template, we don't need to make quality point for each product.

    Step 2 : Create Picking
  • Please do Sales or Purchase to create picking
  • On picking click on Quality Check button
    • Step 3 : Check the Quality
  • Put OK if the test is OK
  • Validate by click on Pass button
