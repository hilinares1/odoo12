Sales Rep Customer Check-in

Please note this module will work only if you are using HTTPS

Customer Location

The module depends on native odoo module Partners Geolocation
The first step is to record the customer location (Lat & Long) as highlighted in yellow in the figure on the left.
The customer location is used later on to calculate the distance between the Sales Rep current location and the customer location

User Check-in

The user(sales person or sales manager) can use the check-in button on the customer form view to record his visit(marked in green)
The module will calculate the distance between the current location of the user and the customer location and create visit record.
You can check the customer visits from the TAB marked in yellow.

Visits Report

Under the reporting menu of sales module you find the visits report menu
The user can group by customer or by user or both
Visits with distance less than 500m are marked in green, visits between 500m and 1.5 Km are marked in yellow and visits greater than 1.5 are marked in red
