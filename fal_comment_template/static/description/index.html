Sale Comment

Create template for sales order line comment

Falinwa Limited

Story behind the features

Odoo behavior : in v11, if you need to give a special comment or a Tailor-made description of your product to your customer, you need to manually do it each time you sell the product to this customer.

Falinwa behavior : The notion of comment template gives you the possibility to address a special comment with a condition on the product and/or the customer.

Module + : by crossing the notion of product and customer, you can really make Tailor-made communication with the customer by using the same product and same catalog with all the customers.

Business case : A customer requires that you write his own internal reference on your quotation for a product which is a catalog product on your side.

Falinwa recommendation :

  • Installation: 20% of the database.
  • Recommandation: upon request, usually when the sales admin is speaking about this recurring issue.

How to use this module

Create Comment Template
  • Go to menu Sales / Configuration / Comment Template
  • Click on Create
  • Put Name, select Partner and Product
  • Click Save
