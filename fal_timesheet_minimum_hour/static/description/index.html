Timesheet Minimum Hour

Set minimum employee weekly working hour

Falinwa Limited

Story behind the features

Odoo behavior : There is no restriction to control the hours of a minimum or maximum hours an employee can put on a timesheet.

Falinwa behavior : Employees have to record their timesheet according to their standard working schedule of the week define in your contract type. The "according to" is configured based on your HR Policy and can be Timesheet hours is equal to Weekly Working Schedule, is less than Weekly Working Schedule, is more than Weekly Working Schedule.

Module + : it ensures that the timesheet are filled according to your HR policy and the contract type of each employee.

Falinwa recommendation :

  • Installation : 100% of the database using timesheet. With = if the company is working without extra hours and with minimum if the company is working with real hours.

How to use this module

Set The Minimum Working Hours
  • Go to menu Employee / Employee Contract
  • Set the minimum working hour
    • Submit Timesheet
  • Submit the timesheet to Manager
  • When the weekly timesheet is below than minimum timesheet working hour, it will show warning
