Employee Contract Template

It makes easy to create an employee contract

Falinwa Limited

Story behind the features

Odoo behavior : Odoo does not propose the notion of contract template for a quick fulfill of the contract data.

Falinwa behavior : the HR manager can write a list of contract template used by the company and attach the official contract type of the company. At any contract modification, the HR assistant is autonomous to create the contract because all the technical features are pre-defined on the contract template.

Module + : the big plus of the module is the automatic generation of the employee contract based on the template.

Falinwa recommendation :

  • Installation : for companies more than 100 users and with HR department.

How To Do

Step 1 : Create Template Contract

  • Go to menu : Employee / Configuration / Employee Contract Template
  • Create the template
    • Step 2 : Create Employee Contract using template

  • Go to menu : Employee / Contract
  • Click on Create button
  • Put Contract Number
  • Select Employee
  • Select Template (It will load template data)
  • Click on Running Stage
