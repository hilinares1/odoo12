How To Do

Step 1 : Set the user

  • After module installation, we need to set the Users access
  • Go to Menu Setting / Users / User
  • Select a users and set Meeting access :
    1. Simple Meeting : default
    2. Advance Meeting : check on extended meeting
    • Step 2 : Create a meeting

  • Go to Menu Calendar
  • Put Meeting Subject
  • Put the attendee to send the invitation
  • Set the time and duration
  • Set the reminder
    • Step 3 : Check the Effective Presence

  • Open Invitation sheet in Meeting Form,
  • Check the Invitation sent (Accepted or Rejected)
  • Give mark of effective presence for attendee
  • Select the Category
    • Step 4 : Validate Meeting

  • When validate the meeting a warning will show
    • Step 5 : Start the Meeting

  • When validate the meeting a warning will show
    • Step 6 : Stop Meeting

  • Click on Stop Button
  • Alocate time for speaker
  • Click on Stop and Alocate time button
