Pipeline Report

This module is used to view the excel report and preview report for all the salespersons.

Step 1:

Click on Settings->Users.Check the boolean if the user is a Salesperson.

Step 2:

Click CRM->Pipeline Report.A wizard will pop up.

Step 3:

Select the salespersons and their stages needed.Click preview button to view the preview report and click generate report button to view the report in excel.
